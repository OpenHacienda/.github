## OpenHacienda

Herramientas abiertas para interactuar con la administración pública española.

### Proyectos

- [**llave**](https://github.com/OpenHacienda/llave) — Implementación open-source del sistema Cl@ve PIN / Cl@ve Móvil de España. CLI y clientes Flutter sobre un núcleo Rust compartido.
- [**puntoBOE**](https://github.com/OpenHacienda/puntoBOE) — Validador de ficheros BOE de la AEAT que funciona 100% en el navegador, sin envío de datos a ningún servidor.
- [**website**](https://github.com/OpenHacienda/website) — Documentación y página del proyecto.

Open tools for interacting with Spanish public administration.

### Projects

- [**llave**](https://github.com/OpenHacienda/llave) — Open-source implementation of Spain's Cl@ve PIN / Cl@ve Móvil authentication system. CLI and Flutter clients sharing a single Rust core.
- [**puntoBOE**](https://github.com/OpenHacienda/puntoBOE) — Browser-based validator for AEAT BOE tax filing formats. Runs entirely in WebAssembly — no data ever leaves your browser.
- [**website**](https://github.com/OpenHacienda/website) — Project documentation and landing page.
